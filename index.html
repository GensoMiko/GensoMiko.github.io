HTML Images

Gambar dapat meningkatkan desain dan tampilan halaman web.
Tag HTML (img) digunakan untuk menyematkan gambar di halaman web.

  • Tag HTML (img) digunakan untuk menyematkan gambar di halaman web.
  • Gambar secara teknis tidak dimasukkan ke dalam halaman web; gambar ditautkan ke web halaman. Tag (img) membuat ruang penyimpanan untuk gambar yang direferensikan.
  • Tag (img) kosong, hanya berisi atribut, dan tidak memiliki penutup tag.
Tag (img) memiliki dua atribut yang diperlukan:
❖ src - Specifies the path to the image
❖ alt - Specifies an alternate text for the image
  • Atribut alt yang diperlukan menyediakan teks alternatif untuk gambar, jika pengguna untuk beberapa alasan tidak dapat melihatnya (karena koneksi lambat, kesalahan dalam src atribut, atau jika pengguna menggunakan pembaca layar).
  • Nilai atribut alt harus mendeskripsikan gambar: Contoh: (img src="img_chania.jpg" alt="Bunga di Chania")
  • Jika browser tidak dapat menemukan gambar, itu akan menampilkan nilai atribut alt Contoh: (img src="wrongname.gif" alt="Bunga di Chania")

Image Size - Width and Height
❖Anda dapat menggunakan atribut gaya untuk menentukan lebar dan tinggi gambar. Contoh (img src="img_girl.jpg" alt="Gadis berjaket" style="width:500px;height:600px;")
❖Atau, Anda dapat menggunakan atribut lebar dan tinggi Contoh (img src="img_girl.jpg" alt="Gadis berjaket" width="500" height="600")
❖Atribut lebar dan tinggi selalu menentukan lebar dan tinggi gambar dalam piksel

Sementara itu, Jika Image yang ingin disematkan berada di sebuah sub folder, anda dapat menautkan image itu seperti berikut (img src="x/x.jpg"). Tetapi, Jika Image berada di suatu website atau server lain, maka cara menautkan Image tersebut adalah dengan melakukan (img src="(Link gambar tersebut)").

HTML Links

Tautan ditemukan di hampir semua halaman web. Tautan memungkinkan pengguna mengklik dengan cara mereka dari halaman ke halaman
● Tautan HTML adalah hyperlink
● Anda dapat mengklik tautan dan melompat ke dokumen lain.
● Saat Anda menggerakkan mouse di atas tautan, panah mouse akan berubah menjadi tangan kecil
● Tautan tidak harus berupa teks. Tautan dapat berupa gambar atau HTML lainnya elemen
Secara default, tautan akan muncul sebagai berikut di semua browser:
❖ Tautan yang belum dikunjungi digaris bawahi dan berwarna biru
❖ Tautan yang dikunjungi digaris bawahi dan berwarna ungu
❖ Tautan aktif digaris bawahi dan merah
Sebuah Contoh Link
Secara default, halaman tertaut akan ditampilkan di jendela browser saat ini. untuk mengubah hal ini, Anda harus menentukan target lain untuk tautan tersebut.
Link diatas tersebut, dan link-link yang tersebar di bagian-bagian sebelumnya menggunakan target _blank yang berarti link akan muncul di tab baru. Target yang lain adalah
❖_self, target yang bersifat default dan akan menampilkan halaman yang ditaut di tab yang sedang dibuka
❖_parent, Membuka dokumen dalam bingkai induk
❖_top, Membuka dokumen di seluruh badan jendela
Link yang dicontohkan diatas merupakan sebuah Link Absolute, sebuah link yang menunjukan alamat web yang penuh, selain Link Absolute ada Link Relative yaitu link local (berada di website yang sama)
Untuk membuat sebuah Link menjadi lebih menarik, anda dapat menggabungkannya dengan tag Button, contoh berikut akan menampilkan hal tersebut dan juga link menuju sebuah alamat email dengan mailto

HTML Tables

Tabel HTML memungkinkan pengembang web untuk mengatur data ke dalam baris dan kolom, Sebuah tabel dalam HTML terdiri dari sel-sel tabel di dalam baris dan kolom. Contoh:

Company Contact Country
Chaldea Goredolf von Musik Germany
Clocktower El-Melloi England
Atlas Institute Sion Eltnam Atlasia Greenland

● Setiap sel tabel ditentukan oleh tag (td) dan (/td)
● td adalah singkatan dari tabel data
● Segala sesuatu di antara (td) dan (/td) adalah konten sel tabel

HTML Forms

Formulir HTML digunakan untuk mengumpulkan input pengguna. Masukan pengguna paling sering dikirim ke server untuk diproses
● Elemen (form) digunakan untuk membuat formulir HTML untuk input pengguna
● Elemen (input), Elemen HTML (input) adalah elemen form yang paling banyak digunakan dan dapat ditampilkan dalam banyak cara, tergantung pada jenis atribut

Beberapa Jenis elemen (html) input
● (input type="text") Menampilkan kolom input teks satu baris
● (input type="radio") Menampilkan tombol radio (untuk memilih salah satu dari banyak pilihan)
● (input type="checkbox") Menampilkan kotak centang (untuk memilih nol atau lebih dari banyak pilihan)
● (input type="submit") Menampilkan tombol kirim (untuk mengirimkan formulir)
● (input type="button") Menampilkan tombol yang dapat diklik
